### 1일 1아티클
#### 요즘IT
###### 바이브 코딩 추천 툴 7가지
**대장 프로덕트**
1. `Claude Code`
  - 터미널 기반 직관적인 접근 방식 + 최신 모델을 활용한 코드 생산성
  - `Subagents`, `Skills`, `Plugins` 등 개발자 니즈를 반영한 지속적인 기능 업데이트
  - 빠르게 커지고 있는 사용자 커뮤니티 + 풍부한 활용 자료
  - 단점 : 반복 작업이나 대규모 프로젝트에서 토큰 한도 발생, 빈약한 프롬프트 설계 시 비용 기하급수적 증가
  - 주요 사용처 : 레거시 코드 베이스의 빠른 파악 및 정리, 리팩토링/테스트 작성/문서화 작업 많은 프로젝트, 개인/소규모 팀에서 AI 동료처럼 활용하여 코드 생산/검증/문서화에 사용
2. `Cursor`
  - 익숙한 IDE 기반 + AI 편집 기능
  - 멀티라인/멀티파일 자동 수정, 프로젝트 전채 리팩토링, 코드 리뷰 보조, 테스트 생성 등 실전 기능 제공
  - 단점 : 대규모 프로젝트에서 맥락 이해의 한계로 인한 잘못된 수정 제안 발생, 구독 비용 부담
  - 주요 사용처 : IDE 흐름을 유지하며 AI로 생산성 극대화, 반복 수정 작업이 많을 때, AI가 코드를 읽고 수정하는 과정을 IDE에서 직관적으로 확인할 때
  
**떠오르는 대안**
1. `Antigravity`
  - 구글이 실험적으로 공개한 AI 에이전트 기반 개발 플랫폼
  - 에이전트를 통해 프로젝트 전체를 조립/운영하는 플랫폼의 성격
  - `Gemini` 기반 + `Firebase`, `Google Cloud` 등 구글 생태계와의 자연스러운 통합에 강점
  - 주요 사용처 : 코드 편집기/에이전트 명령/실행 화면을 하나의 IDE에서 동시에 보기, 구글 생태계 기반의 서비스 구축 시
2. `Codex`
  - `ChatGPT`와 직결되는 웹 기반 AI 코딩 에이전트
  - 자연어 인터페이스에서 즉석 코드 실행까지 가능 (검색 → 복붙 → 테스트 과정 단축)
  - 모바일 환경 지원
  - 주요 사용처 : `ChatGPT`에서 바로 코드 생성 및 실행 시, 모바일 환경에서 간단한 작업 즉시 처리, 간단한 서비스/스크립트/자동화 기능을 빠르게 실험/반복 시
3. `Warp`
  - AI 지원 터미널로, 기존 터미널의 불편 개선
  - GPU 기반 렌더링으로 빠른 속도, `GitHub`/`AWS` 등 다양한 개발 도구와 연동
  - 주요 사용처 : DevOps/백엔드/데이터 작업에서 더 빠르고 편한 터미널 환경, 팀 단위로 명령어/스크립트/자동화 흐름 공유 및 표준화 시
  
**특화 프로덕트**
1. `CodeRabbit`
  - PR 리뷰 병목 및 품질 편차 해결을 위한 AI 리뷰 에이전트
  - `GitHub PR`과 직접 연결되어 자동 리뷰 코멘트/개선 제안/테스트 보완 포인트 생성
  - 변경 파일/기존 문맥 기반 코드 요약/위험 지점 표시/로직 설명 제공
  - 팀 설정을 통한 리뷰 기준 일관성 유지 및 반복적 리뷰 업무 자동화
2. `v0 by vercel`
  - `Vercel`의 AI 기반 UI 코드 생성 도구
  - 초기 화면을 빠르게 제작할 때 발생하는 프론트엔드 병목 해결
  - `Next.js` + `React` 기반 UI 코드 생성
  - 기본 컴포넌트부터 라우트/페이지 구조/상태 흐름까지 초기 화면 자동 구성
  - 실제 프로젝트에서 바로 수정할 만한 형태로 제공
  - 짧은 시간 안에 화면을 보고 결정해야 하는 MVP/프로토타입 환경에서 유용
