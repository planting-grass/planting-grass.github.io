### 1일 1아티클
#### 카카오 테크
###### Kanana-2
**소개**
- 카카오의 차세대 언어모델
- 사용자의 명령 맥락 파악 후 능동적으로 동작하는 Agentic AI 구현에 최적화
  - 도구 호출 능력 및 지시 이행 능력 향상에 초점
- 전반적으로 `Qwen3-30B-A3B-2507` 성능 수준
- 한국어 LLM 성능 평가 지표에서 국내 모델 1위 달성
- 오픈소스로 공개
  
**버전**
1. `Base`
  - 사전 학습 단계 웨이트 포함된 기본 모델
2. `Instruct`
  - Base 모델 기반으로 Post-training 진행, 사용자의 지시 이행 능력 극대화
3. `Thinking`
  - 카카오의 최초 추론 특화 모델
  
**강점**
1. 외부 도구 호출 성능 강화
  - 이전 모델 kanana-1.5-32.5b 대비 3배 이상 개선
2. 지시 이행 능력 고도화
3. 6개 국어 지원
4. 추론형 AI 가능성 확인
5. 대규모 트래픽 환경 응답속도 개선
  - 최신 아키텍처인 MLA, MoE 도입
  - `MLA` : 메모리 압축으로 긴 문맥 효율적 처리
  - `MoE` : 전문가 혼합 모델 구조, 추론 시 필요 파라미터만 활성화
