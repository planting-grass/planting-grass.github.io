#### ITWORLD
###### Gemini 3 API 개선
**개선 사항**
1. 사고 레벨(thinking_level) 도입
  - 모델 응답 생성 전 내부적으로 수행하는 추론의 최대 깊이 조정
  - 개발자는 작업 복잡도에 따라 해당 값 조정 가능
2. 멀티모달 비전 처리 제어 기능 세분화
  - 신규 미디어 해상도(media_resolution) 파라미터를 통해, 이미지/동영상/문서 입력 시 사용 토큰 수 조정
  - 개발자는 시각적 정밀도와 토큰 사용량 간 균형 조정 가능
3. 사고 서명(Thought Signatures) 기능 재도입
  - 모델 내부 사고 과정을 암호화한 데이터 표현
  - 이후 API 호출 시 모델에 재전달하여 Gemini 3이 대화 전반의 추론 연속성 유지 가능
4. 구글이 호스팅하는 도구와 결합 활용
  - 개발자는 구글 검색 연동과 URL Context 기능을 함께 사용 가능
  - 특정 웹페이지에서 실시간 정보를 가져와 데이터를 JSON으로 추출하는 에이전트 구축 시 유용
  - 에이전트가 웹 데이터를 실시간으로 수집/정리/활용하는 복합형 작업을 정교하게 수행 가능하도록 지원
