# 오늘 한 일
- MSW(Mock Service Worker) 학습 및 적용
  - 서비스 워커(Service Worker)를 활용해 네트워크 레벨에서 API 요청을 가로채는 원리 이해
  - 백엔드 API 완성 전에도 프론트엔드 개발을 독립적으로 진행할 수 있는 모킹 환경 구축
  - handlers 설정을 통해 실제 서버 응답과 유사한 모의 데이터(Mock Data) 연동
  - 브라우저와 테스트 환경 모두에서 동일한 모킹 로직을 공유하여 개발 효율성 제고
  - 모든 모킹 코드 작성 시 any 키워드 사용을 지양하고 명확한 타입을 정의하여 안정성 확보
