## margin과 padding

### margin

> 요소의 바깥쪽 여백

- 콘텐츠(content)와 테두리(border) 사이 공간
- 요소 내부 공간
- 배경색(background)이 적용됨

```css
.box {
  padding: 20px;
}
```

### padding

> 요소의 안쪽 여백

- 요소와 요소 사이 간격
- 요소 외부 공간
- 배경색 적용 X

```css
.box {
  margin: 20px;
}
```

### 차이

| 구분           | Padding   | Margin    |
| -------------- | --------- | --------- |
| 위치           | 요소 내부 | 요소 외부 |
| 배경색 적용    | O         | X         |
| 콘텐츠 영향    | O         | X         |
| 요소 크기 영향 | O         | X         |
| 요소 간 간격   | X         | O         |

### 알아야 할 포인트

- box-sizing과의 관계
  - `border-box`
    - padding + border 포함해서 width 계산
  - 기본값(`content-box`)
    - width는 콘텐츠 영역만 padding은 그 밖범위 → padding 추가시 실제 박스 크기 늘어남
- Margin Collapse
  - 세로 방향 margin은 겹쳐짐
  - 더 큰 margin 하나만 적용됨
  ```css
  div {
    margin-top: 20px;
    margin-bottom: 30px;
  }
  ```
  - 위에서 실제 간격 = **30px**
