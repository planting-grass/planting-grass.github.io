## 4. Comparator

### 기본 틀

```java
Arrays.sort(arr, (a, b) -> {return b - a; }); // 내림차순
```

### 문자열 정렬

```java
Arrays.sort(arr, (a,b)-> a.compareTo(b));
```

### 여러 기준 정렬

```java
Arrays.sort(arr, (a,b)->{if(a[0] == b[0]){return a[1] - b[1];
    }return a[0] - b[0];
});
```

## 5. 문자열 숫자 정렬

```java
Arrays.sort(arr, (a,b)-> (b+a).compareTo(a+b));
```
