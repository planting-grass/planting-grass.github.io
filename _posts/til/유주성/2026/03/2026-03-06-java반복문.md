## 1. While 문

- 자바스크립트와 별반 다를게 없다..

```java
package roop;

public class While1_2 {
    static void main(String[] args) {
        int count = 0;

        while(count < 3){
            count++;
            System.out.println("현재 숫자는: " + count);
        }
    }
}
```

```java
package roop;

public class While2_3 {
    static void main(String[] args) {
        int sum = 0;
        int i = 1;
        int endNum = 3;

        while(i <= endNum){
            sum = sum + i;
            System.out.println("i=" + i + " sum=" + sum);
            i++;
        }
    }
}

```

## 2. Do While

- 한번 실행하고 WHILE 문을 실행

```java
package roop;

public class DoWhile2 {
    static void main(String[] args) {
        int i = 0;
        
        do{
            System.out.println("현재 숫자는:" + i);
            i++;
        } while(i < 3);
    }
}

```

## 3. break, continue

- break : 반복문에서 탈출
- continue : 반복문의 조건부분으로 바로 되돌아감

```java
package roop;

public class Break1 {
    static void main(String[] args) {
        int sum = 0;
        int i = 1;

        while(true){
            sum = sum + i;
            if (sum > 10){
                System.out.println("합이 10보다 크면 종료 : i=" + i + " sum=" + sum);
                break;
            }
            i++;
        }
    }
}

```

```java
package roop;

public class Continue1 {
    static void main(String[] args) {
        int i = 1;

        while(i <= 5){
            if (i == 3){
                i++;
                continue;
            }
            System.out.println(i);
            i++;
        }
    }
}

```
